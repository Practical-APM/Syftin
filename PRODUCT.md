# Syftin — Product Context

## Product Purpose

Syftin is a distributed edge-intelligence platform for enterprise web data extraction. Phase 1 delivers structured JSON datasets from whitelisted public domains via a centralized Go worker pipeline, with a buyer dashboard for job tracking and downloads.

## Users

- **Enterprise buyers (Data Controllers):** E-commerce aggregators, registry analysts, job-market analytics teams. Need compliant, schema-accurate datasets without managing scraping infrastructure.
- **Contributors (Phase 2):** College students and homelab operators monetizing idle GPU and residential bandwidth.
- **Platform operators:** Syftin team managing whitelist, schemas, validation, and payouts.

## Brand & Tone

- Precise, enterprise-trustworthy, technically credible
- Not "hacker" or "bypass" language — "consumer-fidelity fetch", "edge redaction", "schema compliance"
- Modern SaaS polish comparable to best-in-class developer tools

## Register

- **Brand:** Marketing landing page, positioning, trust signals
- **Product:** Dashboard, job management, downloads

## Anti-references

- Crypto-mining aesthetic
- Generic purple-gradient AI slop
- Overpromising "unlimited scraping"

## Strategic Principles (Phase 1)

- Whitelist enforced at job submission
- Syftin acts as Data Processor; client is Data Controller
- 98%+ schema compliance on 5 designated domains
- Central LLM API for parsing; edge network deferred to Phase 2
